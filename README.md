# SecureApp-Sprint-StudentPortal

## Project Overview
The Student Result Portal is a secure web-based application developed using Flask. It allows students to view their academic results while enabling administrators to manage and update student marks.  

This project demonstrates the implementation of secure software development practices by identifying and mitigating common web vulnerabilities.

## Security Features Implemented
The application addresses key *OWASP Top 10* vulnerabilities:

-  Broken Access Control (IDOR fixed)
-  Cross-Site Request Forgery (CSRF protection)
-  Clickjacking Protection (Security Headers)
-  Role-Based Authorization (RBAC)
-  Session Management
